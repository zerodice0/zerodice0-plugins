## English

A collection of Claude Code plugins developed by zerodice0.

### Available Plugins

| Plugin | Description |
|--------|-------------|
| [gemini-design-updater](./plugins/gemini-design-updater/) | Safe design update workflow using Gemini Pro with Claude review |

### Installation

```bash
# Add marketplace
/plugin marketplace add https://github.com/zerodice0/zerodice0_claude_plugins.git

# Install plugin
/plugin install gemini-design-updater@zerodice0_claude_plugins
```

### Requirements

- Claude Code CLI
- Git 2.0+
- Bash 4.0+
