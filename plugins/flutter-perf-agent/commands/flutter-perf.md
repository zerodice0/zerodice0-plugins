---
name: flutter-perf
description: Flutter 성능 분석 에이전트 실행 (VM Service 연동)
---

flutter-perf 스킬을 실행합니다.

`flutter run --profile --vm-service-port`와 Dart VM Service를 연동하여 Flutter 앱의 성능을 분석하고, 최적화 제안을 제공하는 워크플로우를 시작합니다.

## 사용 방법

```
/flutter-perf
```

또는 프롬프트에서 직접:
- "Flutter 성능 분석해줘"
- "Flutter 앱 성능 측정해줘"
- "Flutter 성능 보고서 만들어줘"
