# 프롬프트 예시

Gemini Image Generator 스킬 사용을 위한 프롬프트 예시 모음입니다.

## 아이콘 생성

### 기본 아이콘
```
로그인 버튼 아이콘 생성해줘
```

### 스타일 지정
```
미니멀한 스타일의 설정 아이콘 만들어줘, 흰색 배경에 파란색 라인
```

### 상세 스타일
```
Material Design 스타일의 홈 아이콘, 둥근 모서리, 회색 톤, 24x24 크기
```

### 앱 아이콘
```
쇼핑 앱 아이콘 만들어줘, 장바구니 모양, 녹색 그라데이션, 1024x1024 크기
```

---

## 로고 생성

### 텍스트 로고
```
"CodeFlow"라는 텍스트 로고 생성해줘, 모던한 산세리프 폰트, 파란색
```

### 심볼 로고
```
테크 스타트업 로고 만들어줘, 추상적인 기하학 패턴, 보라색과 청록색 조합
```

### 조합 로고
```
커피숍 로고 생성해줘, 커피잔 아이콘과 "Bean Bros" 텍스트 조합
```

---

## 배경 이미지

### 그라데이션 배경
```
그라데이션 배경 이미지 PNG로 생성해줘, 파란색에서 보라색으로, 1920x1080 크기
```

### 패턴 배경
```
물결 패턴 배경 만들어줘, 밝은 회색 톤, 반복 가능한 타일 형태
```

### 추상 배경
```
앱 스플래시 화면용 추상 배경, 부드러운 곡선, 파스텔 톤
```

## UI 요소

### 버튼
```
라운드 버튼 디자인 만들어줘, 그라데이션 효과, 그림자 포함
```

### 카드
```
프로필 카드 UI 요소 만들어줘, 아바타 영역과 정보 영역 구분
```

### 일러스트
```
빈 상태(empty state) 일러스트 만들어줘, 검색 결과 없음을 표현, 귀여운 스타일
```

## 멀티 해상도 생성

### Flutter 프로젝트
```
앱 로고 생성하고 Flutter 프로젝트에 맞게 1x, 2x, 3x 크기로 저장해줘, 경로는 ./assets/
```

### iOS 프로젝트
```
앱 아이콘 만들고 iOS용 @1x, @2x, @3x 크기로 변환해줘
```

### Android 프로젝트
```
앱 아이콘 생성하고 Android drawable 폴더에 맞게 mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi로 저장해줘
```

### 모든 플랫폼
```
앱 로고 만들고 Flutter, iOS, Android, Web 모든 플랫폼용으로 변환해줘
```

## 포맷 지정

### SVG (벡터)
```
로고 아이콘 SVG 포맷으로 생성해줘, 확대해도 깨지지 않게
```

### PNG (래스터)
```
그라데이션이 포함된 배경 이미지 PNG로 생성해줘
```

### WebP (최적화)
```
웹용 배너 이미지 WebP 포맷으로 생성해줘, 파일 크기 최소화
```

## 고급 예시

### 상세 스타일 지정
```
e-commerce 앱용 메인 아이콘 세트 생성해줘:
- 홈, 검색, 장바구니, 프로필, 설정
- Outlined 스타일, 선 두께 2px
- 24x24 기본 크기
- 색상: #333333 (비활성), #007AFF (활성)
- 저장 경로: ./assets/icons/
```

### 반응형 에셋
```
반응형 히어로 이미지 생성해줘:
- 데스크톱: 1920x600
- 태블릿: 1024x400
- 모바일: 640x300
- 주제: 테크놀로지, 미래지향적
- PNG 포맷
```

## 팁

1. **구체적으로 설명하세요**: 색상, 스타일, 크기를 명확히 지정할수록 원하는 결과를 얻기 쉽습니다.

2. **참조 이미지 활용**: 비슷한 스타일의 이미지가 있다면 `@경로`로 참조하면 일관된 스타일을 유지할 수 있습니다.

3. **포맷 선택 가이드**:
   - 아이콘, 로고 → SVG
   - 복잡한 이미지, 사진 → PNG
   - 웹 최적화 → WebP

4. **멀티 해상도**: 앱 개발 시 처음부터 멀티 해상도로 생성하면 나중에 추가 작업이 줄어듭니다.
